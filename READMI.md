# Spotik

Система мониторинга парковочных мест с использованием компьютерного зрения. Проект включает бэкенд на FastAPI, детекцию автомобилей с помощью YOLO, веб-интерфейс и аутентификацию через Google OAuth.

## Основные функции

- Детекция автомобилей в видеопотоке (файл или камера) с помощью YOLOv8.
- Подсчёт количества машин на парковке и свободных мест на основе пересечения линий въезда/выезда.
- Отображение видеопотока в реальном времени (MJPEG) с наложенной информацией.
- Статистика в реальном времени через WebSocket.
- Аутентификация пользователей через Google OAuth и по паролю (регистрация, вход).
- Хранение пользователей в SQLite.
- API документация через Swagger.

## Технологии

- Python 3.11+
- FastAPI
- Uvicorn
- SQLAlchemy
- OpenCV
- Ultralytics YOLOv8
- Authlib (Google OAuth)
- Passlib (хэширование паролей)
- SQLite
